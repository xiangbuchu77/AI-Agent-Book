# AI-Agent-Book

《AI Agent 开发圣经：从 LLM 到 Agent（2026）》

This repository contains the source for an open-source technical book about AI Agent development.

## Book Structure

- 00 Preface
- 01 AI Basics
- 02 Transformer
- 03 Prompt
- 04 RAG
- 05 Agent
- 06 LangGraph
- 07 MCP
- 08 Projects
- 09 System Design
- 10 Interview

## Local Development

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the documentation server:

```bash
mkdocs serve
```

Build the static site:

```bash
mkdocs build
```

Generate the PDF edition:

```bash
python scripts/build_pdf.py
```
