# AI Agent 开发圣经：从 LLM 到 Agent（2026）

这是一本面向开发者的 AI Agent 工程实践书。它从 LLM 基础开始，逐步进入 Prompt、RAG、Agent、LangGraph、MCP、项目实战、系统设计和面试准备。

## Reading Path

1. 先理解 LLM 的输入、输出、上下文和能力边界。
2. 再学习 Prompt 与 RAG，把模型能力接入可控上下文和外部知识。
3. 然后进入 Agent，理解工具调用、记忆、状态、规划和执行循环。
4. 最后通过 LangGraph、MCP、项目实战和系统设计，把能力组合成生产级系统。
